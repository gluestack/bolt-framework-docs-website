# Installing the Global CLI

This document provides installation instructions for the [bolt-cli](https://www.npmjs.com/package/@gluestack/bolt-framework).

## Prerequisites

- Node.js (version 18.0.0 or higher)
- Package Manager (e.g., NPM or Yarn)

## Installation Steps

In order to install [bolt-cli](https://www.npmjs.com/package/@gluestack/bolt) in your system you can either use yarn or npm to install the package.

```bash

# using npm

npm install --global @gluestack/bolt@latest

# using yarn

yarn global add @gluestack/bolt@latest

```

After the installation is complete, you can check if the package is installed correctly by running the following command in your terminal.

```bash

bolt

```

If it's installed correctly it should give the output something like this:

```bash
Usage: Bolt [options] [command]

Bolt CLI tool v0.1.1

Options:
  -V, --version   output the version number
  -h, --help      display help for command

Commands:
  init [options]  Inits the project with bolt.yaml file
  help [command]  display help for command
```
