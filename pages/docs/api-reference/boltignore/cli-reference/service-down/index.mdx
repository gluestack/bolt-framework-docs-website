# service:down

This command helps you to bring down a service running against a bolt project on your host or docker.

```bash
$ bolt service:down --help
Usage: bolt service:down [options] <service-name>

Stops a service available in "bolt.yaml" file

Arguments:
  service-name  service containing "bolt.service.yaml" file and present in
                "bolt.yaml" services
Options:
  -h, --help    display help for command
```
