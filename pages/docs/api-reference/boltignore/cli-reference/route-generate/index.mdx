# route:generate

This command generates nginx configuration into a bolt.nginx.conf file using the ingress object available in your bolt.yaml file.

```bash
$ bolt route:generate --help
Usage: bolt route:generate [options]

Generates "bolt.nginx.conf" file from "bolt.yaml" file's ingress object

Options:
  -h, --help  display help for command
```
