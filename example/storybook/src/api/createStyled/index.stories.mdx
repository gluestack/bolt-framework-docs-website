# createStyled()

You can add custom plugins to resolve the style of your component before it gets to the styled component. createStyled is a function that takes in a array of plugins and returns a function that takes in a component and returns a styled component.

```jsx
import {
  CustomPlugin,
  AnimationPlugin,
  createStyled,
} from '@gluestack-style/react';

const styledCustomized = createStyled([
  new CustomPlugin({}),
  new AnimationPlugin({}),
]);

const StyledView = styledCustomized(
  View,
  {
    w: 100,
    h: 200,
    bg: '$blue500',
  },
  {
    descendantStyle: ['_text'],
  }
);

const StyledText = styledCustomized(
  Text,
  {},
  {
    ancestorStyle: ['_text'],
  }
);

export function App() {
  return (
    <Wrapper>
      <StyledView>
        <StyledText>Hello World</StyledText>
      </StyledView>
    </Wrapper>
  );
}

export default App;
```
